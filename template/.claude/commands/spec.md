---
description: Cria ou atualiza a especificação de uma feature (spec-driven), via skill requisitos-spec.
---

Inicie o desenvolvimento orientado a requisitos para a feature descrita abaixo.

1. Acione a skill `requisitos-spec`.
2. Faça as perguntas necessárias para remover ambiguidade.
3. Produza `specs/<feature>/spec.md` a partir de `specs/_template-spec.md`.
4. Ao final, sugira `/planejar <feature>` como próximo passo.

Descrição da feature: $ARGUMENTS
