Entregável final

Um template de codificação versionável com CLAUDE.md (regras + organização de pastas), 3 subagents (planejador, revisor, testador), 2 skills (requisitos-spec, revisao-codigo), 2 slash commands (/verificar, /commit), settings.json com permissões e hook de formatação, .mcp.json e modelo de spec. Já incluído nesta entrega — basta renomear dot-claude para .claude.

Tempo total estimado: ~6 semanas em ritmo de quem já usa Claude Code no dia a dia.
Sintaxe de skills/subagents/settings evolui entre versões — confirme sempre na doc oficial: code.claude.com/docs
